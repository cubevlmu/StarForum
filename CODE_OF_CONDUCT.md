# Code of Conduct

## Our Pledge

We are committed to making participation in this project a welcoming and respectful experience for everyone.

## Expected Behavior

Participants are expected to:

- Be respectful and inclusive
- Provide constructive feedback
- Focus on improving the project

## Unacceptable Behavior

Examples include:

- Harassment or discrimination
- Personal attacks
- Trolling or inflammatory comments
- Uncensored AI code using in pull request.

## Enforcement

Project maintainers have the right to remove or edit comments, issues, or pull requests that violate this code of conduct.

This Code of Conduct applies to all project spaces including:

- GitHub Issues
- Pull Requests
- Discussions
